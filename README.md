# Corporate Team Dashboard 

Status: Completed 
Date: January 2026

## Project Summary
A full-stack employee management system built with Enterprise Clean Architecture. This project demonstrates advanced frontend-backend integration using a modern tech stack.

###  Tech Stack
Frontend: React.js (Vite), Glassmorphism UI, Recharts (Analytics)
Backend: .NET Core Web API, Repository Pattern, SQL Server
Tools: Swagger UI, Postman, Git

###  Key Features
Visual Analytics: Real-time Pie & Bar charts for team insights.
Advanced Data Grid: Instant Search, Filtering, and CSV Export.
Architecture: Implemented Dependency Injection and Global Error Handling Middleware.
UX: Custom "Liquid Glass" design with Skeleton loading states.
